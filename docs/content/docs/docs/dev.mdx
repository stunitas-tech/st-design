## 로컬 환경 세팅 방법

<Steps>
    <Step>
    
    ### Homebrew 설치
    
    ```bash
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    ```
    
    Brew 버전 확인하여 설치 확인하기
    ```bash
    brew --version
    ```

    </Step>
    <Step>

    ### Git 설치

    ```bash
    brew install git
    ```

    Git 버전 확인하여 설치 확인하기
    ```bash
    git --version
    ```

    </Step>
    <Step>
    ### Node.js 설치

    노드 버전 매니저 설치

    ```bash
    brew install nvm
    ```

    zsh 설정

    ```bash
    mkdir ~/.nvm
    ```
    ```bash
    echo 'export NVM_DIR="$HOME/.nvm"' >> ~/.zshrc
    ```
    ```bash
    echo '[ -s "/opt/homebrew/opt/nvm/nvm.sh" ] && \. "/opt/homebrew/opt/nvm/nvm.sh"' >> ~/.zshrc
    ```
    ```bash
    source ~/.zshrc
    ```

    Node LTS 설치

    ```bash
    nvm install --lts
    ```
    ```bash
    nvm use --lts
    ```

    설치 확인

    ```bash
    node -v
    ```
    ```bash
    npm -v
    ```
    </Step>
    <Step>
    ### bun 설치

    설치

    ```bash
    curl -fsSL https://bun.sh/install | bash
    ```

    설치 확인

    ```bash
    bun -v
    ```
    </Step>
    <Step>
    ### 프로젝트 코드 클론 받기

    workspace 폴더 생성 후에 프로젝트 클론받고 VSCode에서 열기

    ```bash
    git clone https://github.com/stunitas-tech/st-design.git
    ```
    </Step>

    <Step>
    ### 프로젝트 실행

    VSCode에서 Ctrl + `(백틱) 눌러 터미널을 연 후, 아래 명령어 실행하기

    ```bash
    bun run docs:dev
    ```
    </Step>

</Steps>
