## Dangi BI

![Logo Design](/images/docs/foundation/logo/logo_design.png)

공단기의 심벌은 수험생이 꿈을 이루고, 자신만의 진짜 삶에 돌아가기 위해.  
수험 과정을 단기로 만들겠다는 우리의 약속을 단기Dangi의 이니셜 D로 표현합니다.

또한, 단기Dangi 교육을 통해 꿈을 이루는 방향성을 찾고,  
자신의 가치를 발견하며, 꿈을 향해 도전하도록 돕는 공단기의 브랜드 메시지를 함축하고 있습니다.

심벌을 구성하는 세가지 키워드 # Speed # Exact # From start to finish

## Brand Logo

![brand Logo](/images/docs/foundation/logo/brand_logo.webp)

단기 브랜드 로고는 D 심볼과 브랜드명이 조합된 형태로서, 제시된 규칙을 기준으로 조합이 가능합니다.  
이 외의 추가 개발이 필요할 시 협의를 통해 개발하여 적용할 수 있습니다.

단기 브랜드는 로고는 Pretendard B서체를 기반으로 제작되었으며, 가독성을 고려한 시각적 보정을 모두 거쳐 제작되었습니다.
